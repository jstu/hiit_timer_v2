# HIIT Timer Audio Files

This directory contains audio files used by the HIIT timer:

- `end-sound1.mp3` - End sound for rounds 1, 7, 13, etc.
- `end-sound2.mp3` - End sound for rounds 2, 4, 6, 8, 10, 12, etc.  
- `end-sound3.mp3` - End sound for rounds 3, 6, 9, 12, etc.
- `select_denied.mp3` - Countdown anticipation sound (3-2-1)
- `ding_arcade.mp3` - Halfway point notification
- `fog_horn.mp3` - 30-second burn alert
- `mario.mp3` - Random jump sound during active periods
